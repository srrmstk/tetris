# Simple Tetris in React

### Install

Clone project and run `npm i` or `yarn install`


### [Try it](https://srrmstk.github.io/tetris/)

Controls:
- ⬆️ rotate shape
- ⬇️ move shape down
- ⬅️ ➡️ move shape horizontally
